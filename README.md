# 📊 Retail Sales Performance Dashboard

🚀 Beginner to Intermediate Data Analytics Project  

---

## 📌 Project Overview
This project focuses on analyzing retail sales data using SQL and visualizing insights through an interactive Power BI dashboard.  
The goal was to derive meaningful business insights and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing products   
- Understand revenue distribution across product categories  
- Build an interactive dashboard for business insights  

---

## 🛠️ Tools & Technologies Used
- SQL (PostgreSQL) – Data creation, transformation, and analysis  
- Power BI – Data visualization and dashboard development  

---

## 📂 Dataset Details
The dataset was created using SQL and contains 60 records with the following fields:

- Employee_ID  
- Name  
- Product_Type  
- Product_Name  
- Price  
- Quantity_Sold  
- Sales_Amount  

---

## 📈 Key Metrics (KPIs)
- Total Revenue: 9,48,000  
- Total Orders: 60  
- Total Quantity Sold: 205  
- Average Order Value: 15,800  

---

## 📊 Dashboard Features
- KPI cards using DAX (Revenue, Orders, Quantity, AOV)  
- Revenue analysis by product category  
- Customer-wise sales performance  
- Interactive filters (Product Type, Customer Name)  
- Visualizations including bar chart, pie chart, and table  

---

## 🧠 Key Insights
- Electronics category generates the highest revenue  
- A small group of customers contributes significantly to total sales  
- Clothing category has the lowest contribution  
- Sales distribution is uneven across product categories  

---
